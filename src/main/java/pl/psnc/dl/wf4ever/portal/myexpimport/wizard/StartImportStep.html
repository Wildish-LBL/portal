This wizard allows you to copy your resources (workflows/files/packs) from myExperiment to dLibra, aggregating them as a new Research Object (RO).

Annotations. The following annotations will be made automatically about each imported resource:

  • Source - a reference to the original myExperiment resource
  • Contributor - a reference of the myExperiment owner
  • Title - as in myExperiment
  • Description - as in myExperiment

The wizard has the following steps:

  1. Start.
  2. Select resources: files, workflows or packs.
  3. Confirm RO identifier.
  4. Create the Research Object in RODL, and copy selected resources from myExperiment.
