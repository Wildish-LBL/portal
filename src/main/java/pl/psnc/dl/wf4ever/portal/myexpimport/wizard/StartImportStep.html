This wizard allows you to copy myExperiment resources (workflows/files/packs) to RODL, aggregating them as a new Research Object (RO).

Annotations. The following annotations will be made automatically about each imported resource:

  • Source - a reference to the original myExperiment resource
  • Contributor - a reference of the myExperiment owner
  • Title - as in myExperiment
  • Description - as in myExperiment
