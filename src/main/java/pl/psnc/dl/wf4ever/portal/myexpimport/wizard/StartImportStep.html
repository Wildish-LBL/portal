This wizard allows you to copy your resources (workflows/files/packs) from myExperiment to dLibra, aggregating them as a new Research Object (RO).

Note: Since the actual structure of RO and its metadata is still under development, for every imported resource its accompanying RDF file will also be transfered to make sure that the metadata will be available for processing in the future.

The wizard has the following steps:

  1. Start.
  2. Choose which workspace you want to import to.
  3. Select resources: files, workflows or packs.
  4. Confirm RO identifier.
  5. Create the Research Object in RODL, and copy selected resources from myExperiment.
  6. Summary.
